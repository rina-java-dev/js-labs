# Лабораторная работа №3
## Основы работы с массивами, функциями и объектами в JavaScript

---

## Цель работы

Изучить методы работы с массивами и функциями в JavaScript, а также применить их для анализа данных транзакций.

---

## Задание

Разработать консольное приложение для анализа транзакций, реализовав набор функций для обработки массива объектов.

---

## Структура проекта

Проект разделён на три файла:

- `transactions.js` — содержит массив транзакций
- `main.js` — содержит все реализованные функции
- `test.js` — содержит тестирование функций

---

## Ход работы

### 1. Создание массива транзакций

Был создан массив объектов `transactions`.

Каждый объект содержит следующие поля:

- `transaction_id` — уникальный идентификатор
- `transaction_date` — дата транзакции
- `transaction_amount` — сумма
- `transaction_type` — тип (debit / credit)
- `transaction_description` — описание
- `merchant_name` — название магазина
- `card_type` — тип карты

---

### 2. Реализация функций

Были реализованы следующие функции:

- `getUniqueTransactionTypes()` — уникальные типы транзакций
- `calculateTotalAmount()` — сумма всех транзакций
- `calculateTotalAmountByDate()` — сумма по дате
- `getTransactionsByType()` — фильтр по типу
- `getTransactionsInDateRange()` — фильтр по диапазону дат
- `getTransactionsByMerchant()` — фильтр по магазину
- `calculateAverageTransactionAmount()` — среднее значение
- `getTransactionsByAmountRange()` — фильтр по сумме
- `calculateTotalDebitAmount()` — сумма дебетовых операций
- `findMostTransactionMonth()` — месяц с наибольшим количеством транзакций
- `findMostDebitTransactionMonth()` — месяц с наибольшим числом дебетовых
- `mostTransactionTypes()` — какой тип преобладает
- `getTransactionsBeforeDate()` — транзакции до даты
- `findTransactionById()` — поиск по ID
- `mapTransactionDescriptions()` — список описаний

---

### 3. Используемые методы JavaScript

В работе были использованы следующие методы массивов:

- `map()` — преобразование данных
- `filter()` — фильтрация
- `reduce()` — агрегирование
- `find()` — поиск элемента
- `Set` — получение уникальных значений

---

### 4. Тестирование

Все функции протестированы в файле `test.js`.

Проверены следующие случаи:

- работа с полным массивом
- работа с пустым массивом
- работа с массивом из одной транзакции

Результаты выводятся в консоль.

## Контрольные вопросы

1. Какие методы массивов можно использовать для обработки объектов в JavaScript?
map()
filter()
reduce()
find()
forEach()
2. Как сравнивать даты в строковом формате в JavaScript?

Даты преобразуются в объект Date, после чего их можно сравнивать:

new Date(date1) > new Date(date2)
3. В чем разница между map(), filter() и reduce()?
map() — преобразует каждый элемент массива и возвращает новый массив
filter() — возвращает элементы, соответствующие условию
reduce() — сводит массив к одному значению (например, сумма)
