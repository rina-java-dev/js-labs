## Контрольные вопросы
1. Чем отличается var от let и const?
var имеет функциональную область видимости
let имеет блочную область видимости
const запрещает изменение значения
2. Что такое неявное преобразование типов?
Это автоматическое приведение типов данных JavaScript во время выполнения операций.
3. В чём разница между == и ===?
== сравнивает значения с приведением типов
=== сравнивает и тип, и значение
