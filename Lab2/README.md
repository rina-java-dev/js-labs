# Лабораторная работа №2
## Реализация методов работы с массивами

---

## Цель работы

Изучить принципы работы с массивами в JavaScript и реализовать основные методы обработки массивов без использования встроенных функций.

---

## Используемые инструменты

- Visual Studio Code
- Node.js
- Консоль браузера

---

## Ход выполнения работы

---

### Шаг 1. Подготовка среды

- Установлен Visual Studio Code
- Установлен Node.js
- Создан файл script.js
- Код выполнялся через Node.js и браузерную консоль

---

### Шаг 2. Реализация базового вывода массива

Были реализованы функции:
- printArray
- printArray1

Они выводят элементы массива в консоль в разных форматах.

Результат:
Каждый элемент массива выводится с индексом.

---

### Шаг 3. Реализация функции forEach

Реализована функция forEach, которая:
- принимает массив и callback
- проходит по массиву с помощью цикла for
- вызывает callback для каждого элемента

Особенность:
Функция не возвращает значение (undefined).

---

### Шаг 4. Реализация функции map

Реализована функция map, которая:
- создает новый массив
- применяет callback к каждому элементу
- сохраняет результат в новый массив

Результат:
Получен новый массив преобразованных значений.

---

### Шаг 5. Реализация функции filter

Реализована функция filter, которая:
- проверяет каждый элемент через callback
- добавляет элементы в новый массив, если условие истинно

Результат:
Получен массив, содержащий только подходящие элементы.

---

### Шаг 6. Реализация функции find

Реализована функция find, которая:
- возвращает первый элемент, удовлетворяющий условию
- прекращает выполнение после нахождения элемента

Результат:
Возвращается один элемент или undefined.

---

### Шаг 7. Реализация функции some

Реализована функция some, которая:
- проверяет, есть ли хотя бы один элемент, удовлетворяющий условию
- прекращает выполнение при первом совпадении

Результат:
Возвращает true или false.

---

### Шаг 8. Реализация функции every

Реализована функция every, которая:
- проверяет, удовлетворяют ли все элементы условию
- прекращает выполнение при первом несоответствии

Результат:
Возвращает true или false.

---

### Шаг 9. Реализация функции reduce

Реализована функция reduce, которая:
- накапливает значение через аккумулятор
- обрабатывает элементы массива последовательно

Результат:
Возвращается одно итоговое значение.

## Контрольные вопросы

1. В чем преимущества колбэков?  
   Позволяют передавать поведение как параметр и делают код гибким.

2. Какие проблемы могут возникать?  
   Ошибки типов, сложность чтения, вложенность.

3. Как реализовать методы без встроенных функций?  
   С помощью цикла for, проверки условий и работы с массивом вручную.
